# 🍞 Hleb Duco Web Monitoring
> Мощный и минималистичный веб-мониторинг для майнеров Duino-Coin в реальном времени.

<p align="center">
  <img src="images/Hleb.png" alt="Logo" width="150" style="border-radius: 50%;">
</p>

---

### 🚀 Основные возможности
* 📊 **Живые графики:** Отслеживание хешрейта в реальном времени.
* 💰 **Мультивалютность:** Мониторинг баланса в **DUCO** и автоматический пересчет в **USD**.
* ⚡ **Скорость:** Легкий интерфейс, созданный при поддержке ИИ.
* 📈 **История:** Визуализация статистики воркеров.

### 🛠 Как использовать
1. Склонируйте репозиторий: `git clone https://github.com/твой-ник/твой-репозиторий.git`
2. Откройте `index.html` в вашем браузере.
3. Введите свой никнейм Duino-Coin и наслаждайтесь мониторингом!
